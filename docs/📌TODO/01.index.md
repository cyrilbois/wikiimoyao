## 微服务

微服务将单一应用程序作为由众多小型服务构成之套件加以开发的方式，其中各项服务都拥有自己的进程并利用轻量化机制（通常为 HTTP 源 API）实现通信。
![](https://static01.imgkr.com/temp/735e7feb11e64edca87ee96114777d93.png)

软件部署方式需要建立在容器上。
